DINOQUEST
=========

To run:

  If you use RVM, update to the latest version of RVM, install and use Ruby 2.1.3
    $ rvm get head
    $ rvm install 2.1.3
    $ rvm use 2.1.3

  Install Gosu gem
    $ gem install gosu

  Clone this repo
    $ git clone https://github.com/cherimarie/dinoquest.git

  Cd into the repo, then run the game!!
    $ cd dinoquest
    $ ruby dinoquest.rb
